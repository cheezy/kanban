## Video Concept

Quick tour of Stride's human-only features: drag-and-drop, visual board customization, manual task creation with forms, instant column editing. Demonstrates that humans and AI have complementary, not competing, capabilities.

**Key Message:** "Humans bring intuition and flexibility. AI brings automation and speed. Together, unstoppable."

## Recording Instructions

### Segment 1: Drag-and-Drop (0:03-0:09)

**Setup:**
- Have task card visible in Ready column
- Position cursor near task
- Take a breath (smooth movement is key)

**Action:**
1. Hover over task (0.5s)
2. Click and hold
3. Drag to Doing column (smooth arc, 1s)
4. Drop
5. Hold for reaction animation (0.5s)

**Recording:**
```
- Start screen recording
- Perform drag-and-drop
- Hold final frame 2 seconds
- Stop recording
```

### Segment 2: Create Task (0:09-0:15)

## Post-Production

### Editing Workflow

1. **Import all segments**
   - Drag-and-drop clip
   - Task creation clip
   - Column addition clip
   - Comment clip
   - Split-screen clip

2. **Trim to exact timing**
   - Each segment to precise duration
   - No dead time between actions
   - Quick cuts for energy

3. **Add cursor effects**
   - Highlight cursor with glow/ring
   - Motion trail on drag-and-drop
   - Click ripple effects

4. **Add annotations**
   - Text labels appear just after action starts
   - Fade in/out: 0.2s
   - Clear, large text

5. **Add transitions**
   - Quick cuts (no fades)
   - Or 0.1s crossfade for smoothness
   - Fast-paced editing

6. **Add voiceover**
   - Sync precisely
   - Fast-paced delivery
   - Enthusiastic tone

7. **Add background music**
   - High energy throughout
   - Start loud, maintain energy
   - Quick fadeout at 0:29

8. **Add end card**
   - Split-screen with text overlay
   - Fade in at 0:27
   - Hold until 0:30
   - Fade to black

### Visual Enhancements

**Cursor Enhancement:**
- Add subtle glow effect

## Technical Notes

### LiveView Animations

**Stride uses Phoenix LiveView:**
- Real-time updates via WebSocket
- Smooth CSS transitions
- No page refreshes

**Capture tips:**
- Record at 60fps to capture smooth animations
- Clear browser cache before recording
- Ensure good internet connection (WebSocket stability)
- Test all interactions before final recording

### Cursor Highlighting

**Tools:**
- **macOS:** KeyCastr, MousePos√©
- **Windows:** Carnac, PointerFocus
- **Linux:** key-mon, screenkey

**Settings:**
- Cursor size: 150% of normal
- Click ripple: Subtle, blue or brand color
- Trail: Light, fades quickly

### Keyboard Shortcuts

**Show in video:**
- `Cmd+K` (or `Ctrl+K`): Quick task creation
- `Esc`: Close modals
- Arrow keys: Navigate tasks
- `/`: Search

**Don't show:**
- Complex shortcuts (keep it simple)
- Developer tools
- Browser shortcuts
